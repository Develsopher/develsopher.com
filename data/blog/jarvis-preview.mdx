## Jarvis 첫 번째 이야기

나는 `Chat GPT`를 좋아한다.

업무를 진행하면서 궁금한것이 발생하면 GPT에게 물어보고 GPT가 답변을 해주는 동안 비동기적으로 나는 다른 작업을 한다.

가끔은 정말 이녀석이 내 업무를 도와주는 비서같다는 생각을 하였다.

사실 많은 사람들은 AI기술을 정말 잘 활용하는 것 같다. 나는 단순히 질문을 하고 답변을 받는 정도지만 이미지를 생성한다거나, 음성(비디오)등등 다양한것들을 활용하고 있는 사람들이 많다.

이번 기회에 나도 제대로 한번 활용해보자는 생각이 들었고 내가 원하는 형태의 서비스를 모두 제공해주는 `Jarvis`를 만들기로 결심하였다.

즉 AI api를 통해서 **내가 원하는 서비스를 모아서 제공해주는 대시보드**를 만들어보자는 것이다.

## 전체적인 프로젝트 흐름

- 내가 원하는 기능들을 집약해놓은 대시보드를 만든다.
  - 채팅, 이미지생성, 비디오생성, 코드생성 등등
- AI api를 연동해서 각 기능들을 구현한다.
- 해당 대시보드를 Public하게 유지하면 API수많은 요청으로 나는 거지가 될 수 있기에 로그인 기능을 만든다.
- 결제 연동 기능을 추가해서 무료로 사용할 수 있는 횟수를 제한한다. (챌린지)

### 사용할 기술들

- Next JS (app router 방식) + Typescript
- Shadcn UI (Tailwind css)
  - 단순한 CSS 라이브러리가 아니다. Tailwind CSS 기반으로 재사용 가능한 컴포넌트 컬렉션이다.
  - 높은 수 준으로 커스터마이징 가능하고 무엇보다도 가볍기때문에 요즘에 정말 애용중이다😋.
- Clerk (인증 및 사용자 관리를 위한서비스) [공식사이트](https://clerk.com/)
  - 백엔드없이 진행되는 프로젝트이기에 해당 무료 솔류션을 사용하려고 한다.
  - 이전 프로젝트에서 몇번 사용해봤는데 생각보다 편리했다. (Docs가 정말 잘 되어있다.)
- 결제 연동
  - Stripe (사실 한국에서 정식 서비스를 지원해주지 않는다...)
  - 방법은 만들면서 찾아보자.

### 페이지 구성

- 랜딩페이지 (홈)
  - 해당 서비스에 대한 소개(홍보)가 필요할 것 같다.
- 대시보드
  - React를 사용해서 만들던 Admin 대시보드처럼 직관적으로 만들고자한다.
- 사이드바
  - 각 기능들을 빠르게 이동할 수 있도록 만들어야겠다.

정리해보니 엄청 간단하지만 재밌는 프로젝트가 될 것 같다.
